# Ardor Calculator
Ardor计算器是Ardor.App团队打造的实用加密数据存储&&计算器移动应用程序，本应用程序特点：

1.安全：数据本地存储采用AES-CBC-256高强度加密算法进行数据加密。

2.绿色：只需存储权限，无任何其他隐私权限申请（甚至网络访问权限都没有），不会收集和传输用户任何数据。

3.开源：经得起审计，无后门，放心使用，如有需要，可进行二次定制开发，自行编译。

# 构建命令：
flutter build apk
